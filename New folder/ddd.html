How to Use with Your Own API Key

To use this chatbot in your own application or IDE:

  1. Sign up for an account at Puter.com
  2. Navigate to your account settings to obtain an API key
  3. Replace YOUR_PUTER_API_KEY in the code with your actual API key
  4. Uncomment the puter.init() code block
  5. For direct API calls without the Puter.js library, uncomment Method 2 and comment out Method 1

Note: The Puter.com API may have usage limits or require a paid subscription for extensive use.
